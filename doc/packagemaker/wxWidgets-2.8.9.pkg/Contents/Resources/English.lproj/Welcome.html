Welcome to the wxWidgets for Mac OS X Installer

wxWidgets is a mature open-source cross-platform C++ GUI framework for Mac OS, Unix, Linux, Windows. It can make use of a variety of native widget sets as well as its own widget set: Mac OS, GTK+, Motif, WIN32. wxWidgets will even run on embedded systems using Linux and X11.

http://www.wxwidgets.org/

This installer guides you through the steps necessary to install wxWidgets 2.8.9 for Mac OS X. To get started, click Continue.
